About

Skinwalkers is the latest game concept from Zeroic Games and designer, Taran Kratz. This mobile app represents the second draft of the application companion for the Skinwalkers game.

This application was bootstrapped with the Yeoman workflow, the jQuery Mobile framework, and HTML5 Boilerplate.
